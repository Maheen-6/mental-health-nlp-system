# Mental Health Detection System (NLP)

## Project Description
This project detects mental health status from text using NLP and Machine Learning.

## Classes
- Normal
- Depression
- Suicidal

## Dataset
Custom dataset with ~53,000 records

## Methodology
- Text Cleaning
- Stopword Removal
- TF-IDF Vectorization
- Logistic Regression Model

## Accuracy
~82%

## Model Type
Supervised Machine Learning (NLP Classification)
